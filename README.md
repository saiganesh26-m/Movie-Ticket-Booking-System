# Movie Ticket Booking System

## Overview

This project is a Python-based Movie Ticket Booking Backend Prototype developed using Object-Oriented Programming (OOP) concepts. The system simulates movie ticket booking operations such as seat allocation, booking management, and seat availability tracking.

## Features

* View available seats
* Book movie tickets
* Real-time seat allocation
* Prevent duplicate bookings
* Object-Oriented Design
* Simple and user-friendly interface

## Technologies Used

* Python
* OOP Concepts
* Exception Handling
* SQL (Database Design)

## Project Structure

Movie-Ticket-Booking-System/

├── main.py

├── database.py

└── README.md

## How to Run

1. Install Python 3.x
2. Download the project files
3. Open the project folder in VS Code
4. Run the following command:

python main.py
